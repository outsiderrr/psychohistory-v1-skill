# Character Card Schema & Validation Rules

> This document defines the strict data standard for all character cards used in Psychohistory.
> Every card — whether from the official library, generated by Nuwa CLI, or pasted by a user in chat — MUST pass this validation.

---

## 2. Validation Rules (Human-Readable)

Every character card must pass ALL of the following checks:

### Required Fields Check
| Field | Rule |
|-------|------|
| `card_version` | Must be `"1.0"` |
| `agent_id` | Lowercase alphanumeric + hyphens only |
| `name` | Non-empty string |
| `role` | Non-empty string |
| `source.type` | One of: `official`, `nuwa-skill`, `user`, `system-derived` |
| `mental_models` | Array of 2-10 items, each with `id`, `name`, `description` |
| `decision_heuristics` | Array of 2-12 items, each with `id`, `name`, `description` |
| `concession_triggers` | At least 1 item with `id` and `description` |
| `red_lines` | At least 1 item |
| `honesty_boundaries` | At least 1 item |

### Quality Checks
| Check | Pass Criteria | Fail Signal |
|-------|--------------|-------------|
| Mental model depth | Each description ≥ 20 characters | One-word descriptions |
| ID format | `mm-01`, `dh-01`, `ct-01` pattern | Random strings |
| Honesty boundaries | Lists specific limitations | Only says "not the real person" |
| No contradiction | Mental models don't directly contradict each other without explanation | Logical inconsistencies |

### Validation Error Messages

When validation fails, the system should output:

```
⚠️ Character card validation failed for [name]:
- [FIELD]: [specific error]
- [FIELD]: [specific error]
Please fix the above issues and resubmit.
```

The smallest card that passes validation:

```json

  "agent_id": "example-leader",
  "name": "Example Leader",
  "role": "Head of State",
  "source": {
    "type": "user",
    "created_at": "2026-04-13"
  },
  "mental_models": [
    {
      "id": "mm-01",
      "name": "Survival First",
      "description": "All decisions are filtered through regime survival as the top priority, even at the cost of economic growth."
    },
    {
      "id": "mm-02",
      "name": "Zero-Sum Worldview",
      "description": "Views international relations as fundamentally competitive — one side's gain is the other's loss."
    }
  ],
  "decision_heuristics": [
    {
      "id": "dh-01",
      "name": "Domestic Audience First",
      "description": "Before any foreign policy decision, evaluate how it plays to the domestic audience."
    },
    {
      "id": "dh-02",
      "name": "Escalate to De-escalate",
      "description": "When cornered, escalate the situation to create new bargaining leverage before seeking a deal."
    }

    {
      "id": "ct-01",

    }
  ],
```
